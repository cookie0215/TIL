# 파이썬 설치하기

### 1. python 사이트에서 파이썬을 설치한다.
파일 다운로드 후, 설치 할 때 반드시 'Add python.exe to PATH'를 체크해야 한다.

### 2. VSCODE 에서 파이썬에 관련된 extension을 설치한다.
- Python
- Python  Extension Pack


### 3. VSCODE에서 터미널을 연다
- 단축키 : ctrl + [ ` ]


### 4. 터미널에 아래 처럼 입력한다.
```
$ python
```
입력 후 [ >>> ] 표시가 잘 나타난다면, 정상적으로 파이썬이 설치되었음을 알 수 있다. 
(터미널이 파이썬 환경으로 구축된 것)

```
$ exit()
```
터미널에서 파이썬 환경을 빠져나올 때 사용
